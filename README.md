This repository contains a comprehensive analysis and predictive modeling project designed to optimize the matching process between loan applicants and lending firms to maximize Revenue per Application. By leveraging Logistic Regression and Random Forest algorithms, this project identifies key determinants of loan approvability and calculates the expected value of each application to drive significant business growth.

.pptx file is the final deliverable
R code (.RMD) is basic data cleaning and multiple regression
KMeansandRandomForest.py contains machine learning models (K-means commented out) to discover insights in the data
AdditionalRevenueCalculationAlgorithm.py contains a back-tested logistic regression solution
